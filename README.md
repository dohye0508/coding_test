# Algorithm Coding Test Viewer 🚀

코딩 테스트를 대비한 다양한 알고리즘(동적 계획법, 그래프 탐색 등) 파이썬 템플릿들을 깔끔한 UI로 모아볼 수 있는 뷰어입니다.

## ✨ 기능
- **다크모드 / 라이트모드 지원**: 우측 상단의 토글 버튼을 통해 테마 변경 가능
- **Glassmorphism UI**: 깔끔하고 트렌디한 디자인
- **코드 및 주석 분리 렌더링**: 파이썬 파일 상단의 `'''` 주석을 자동으로 파싱하여, 설명과 [입력/출력 예시]를 코드와 분리해서 예쁘게 보여줍니다.
- **클립보드 복사**: 클릭 한 번으로 코드를 복사할 수 있습니다.

## 🛠 사용 방법 (시작하기)

이 저장소를 Clone 한 직후 특별한 설치 없이 바로 사용이 가능합니다!

1. 저장소를 Clone 받습니다.
```bash
git clone <저장소 주소>
cd coding_test
```
2. 폴더 안에 있는 `index.html` 파일을 크롬 등 웹 브라우저로 엽니다. 
   - (윈도우의 경우 `index.html` 더블 클릭)
3. 왼쪽 트리 메뉴에서 원하는 알고리즘 파일을 선택해 코드와 상세 설명을 확인하세요.

---

## 👩‍💻 알고리즘을 추가/수정하고 싶다면?

자신만의 새로운 알고리즘 파이썬 파일을 추가하거나 내용을 수정한 경우, 브라우저 화면(`index.html`)에 변경 사항을 반영하려면 **데이터 갱신 스크립트**를 실행해야 합니다.

1. 새로운 `.py` 파일을 폴더 구조에 맞게 추가하거나 내용을 수정합니다. (작성 규칙은 `프롬프트.txt` 참고)
2. 터미널(CMD/PowerShell)을 열고 아래 명령어를 실행합니다.
```bash
python generate_data.py
```
3. `data.js` 파일이 성공적으로 갱신되었다는 메시지가 뜨면, `index.html` 화면을 새로고침하여 변경 사항을 확인합니다.
